# 虚空殖民地建筑素材说明

本文件夹存放虚空殖民地场景所使用的建筑纹理素材，风格统一为锈蚀金属、生锈铁板与工业管线的后启示录工业风，主色调为深灰蓝色金属板 + 橙红铁锈 + 暗铜色管道。
以下对每张素材做尺寸、画面内容、以及在地图布局/拼接时的使用建议进行整理，方便后续编写结构生成代码或放置逻辑时查阅。

> 换算参考：Terraria 单格 Tile = 16x16 像素。下文括号内给出了大致的 Tile 尺寸。

---

## 1. CoreVoidLab.png（核心虚空实验室）

尺寸：1162 x 756（约 72.6 x 47.25 Tile）

画面内容：
- 巨型核心建筑，屋内正中为三根垂直的铜线圈能量柱，下方连着一块带有 "Colossare Resonance" 字样的主控大屏
- 两侧分别设有一块小型状态显示屏与操作台，右侧可见一个紫色座椅与操作终端，整体表现为主控/指挥中心
- 建筑下半部分为带楼梯和栏杆的夹层结构，底部中央是大号的圆形探测器或反应舱
- 建筑底座向左右延伸出两段铆接桁架状的外挑平台，平台下有支撑梁结构

使用建议：
- 作为整个虚空殖民地的核心地标建筑，每个殖民地场景只生成一座，位于地图中心或制高点
- 由于素材自带向外延伸的左右桁架平台，应优先与 `ConnectionBridgeSegment` 接驳：两端底部的桁架端点即为桥梁对接口
- 建议以建筑中央能量柱所在的列作为对齐锚点，其他附属建筑（能源站、分析室）向两侧延伸对称分布
- 由于图素尺寸较大，建议在结构表（StructureTile）中单独存一张，放置时一次性贴图

## 2. EnergyControlStation.png（能源控制站）

尺寸：482 x 272（约 30.1 x 17 Tile）

画面内容：
- 四个带闪电符号的圆柱形蓄能罐，前后两排错落排列，罐体之间以铜色管道互联
- 右侧带一面黑色带指示灯（红/黄/绿）的操作面板，面板下方有一扇小型维护舱门
- 顶部和侧面有延伸出的铜管接口，暗示可向外输出能量

使用建议：
- 功能定位为能源节点，负责给核心实验室与其他设施供电
- 顶部与右上方管道端口适合对接 `TubularConnectorTunnel` 管道，向核心实验室输送能量
- 建议放置在核心实验室的左右两翼，且略低于核心层，用桥梁 + 管道双重连接
- 因为底部是平整基座，可以直接落在地表或桁架平台上

---

## 3. MidSizeMaterialAnalysisLab.png（中型物料分析实验室）

尺寸：766 x 368（约 47.9 x 23 Tile）

画面内容：
- 左侧为一座带炉膛火光的熔炼/烧结炉，上方接有排烟通道和冷凝罐
- 中部为一组带玻璃罩的分析柱（类似离心或培养容器），柱内透出铜色金属结构
- 右侧为带图表曲线显示屏的数据终端，终端旁有小型仪表与操作台
- 屋顶有方形烟囱/排气口，地面最右端长着一小丛装饰植物

使用建议：
- 功能定位为中等规模的材料处理与数据分析车间，介于核心实验室与小型设施之间
- 建筑底座为平直铆接地板，左右两端均可直接与桥梁段或同高度的其他建筑拼接
- 建议与能源控制站相邻放置（分析实验需要供能），利用管道从能源站接入左侧或顶部
- 顶部的方形烟囱结构可作为视觉上的纵向变化点，避免天际线过于平直

## 4. ObservationPostTelescope.png（观测哨 / 望远镜台）

尺寸：262 x 310（约 16.4 x 19.4 Tile）

画面内容：
- 方形石砌/金属砌的塔楼，塔顶为一座可俯仰转动的大口径铜色望远镜（三脚架支撑）
- 塔身正面有一扇带内部管道的入口门，门左上方挂着一块带齿轮图案的铭牌
- 塔基两侧有外露的铜管与少量藤蔓植物装饰，整体偏矮塔造型

使用建议：
- 功能定位为前哨/观测点，建议生成在殖民地外围高地或悬崖边缘，用于远望虚空深处
- 体量最小且纵向高于宽度，非常适合作为殖民地的边界地标，不参与主桥梁连接
- 可多座分散放置（2~3 座），以此限定殖民地范围
- 放置时请让塔底与地形贴合，塔顶望远镜不要被其他结构遮挡

## 5. ConnectionBridgeSegment.png（连接桥梁段）

尺寸：552 x 150（约 34.5 x 9.4 Tile）

画面内容：
- 一段水平的 X 形铆接桁架桥，上下各有一条厚重的锈蚀铁质主梁
- 两端的主梁端面带有两颗大号铆钉，作为与其他桥段或建筑对接的接口特征
- 桥身中段为可踩踏的实心底梁，适合作为通行甲板

使用建议：
- 这是**可平铺拼接**的通用桥段，用于将几座大型建筑横向串联成一条空中走廊
- 拼接方式：将当前桥段的右端铆钉接口对齐下一个桥段或建筑的左端接口，保持桥面底梁 Y 坐标严格一致
- 在结构生成时应作为独立 "Segment" 重复使用，长度通过复制段数来控制，不建议拉伸单张贴图（会破坏铆钉细节比例）
- 可与 `CoreVoidLab` 底部外挑桁架、`EnergyControlStation`、`MidSizeMaterialAnalysisLab` 的平直基座无缝对接
- 如需拐弯或跨越高度差，应另做转接件，本段仅表现直线水平段

---

## 6. TubularConnectorTunnel.png（管状连接通道）

尺寸：385 x 122（约 24 x 7.6 Tile）

画面内容：
- 一段水平的圆柱形玻璃管道，管身分成三节，由两道铆接金属箍分节
- 管道两端各有一个较粗的铜铁复合接头，接头上下有管路分支
- 玻璃管内可见到淡淡的高光反光，暗示为加压或有内部流体/能量传输

使用建议：
- 功能定位为人员 / 物料 / 能量输送管道，可视为与 `ConnectionBridgeSegment` 并行存在的**第二套连接系统**
- 同样为**可平铺拼接**的段落素材，拼接时以两端铜色接头为对齐点，严格保持水平
- 建议走在桥梁上方或下方（例如桥梁下方挂管、桥梁上方并行），形成 "桥 + 管" 的双层工业感廊道
- 与 `EnergyControlStation` 的顶部/侧面接口对接最为契合，也可接入 `MidSizeMaterialAnalysisLab` 左上方
- 段数同样通过重复复制控制长度，不要水平拉伸贴图

---

## 整体布局建议

1. 中心放置 `CoreVoidLab` 作为主建筑与对称轴
2. 其两侧以 `ConnectionBridgeSegment` 延伸空中走廊，走廊上按顺序串联 `MidSizeMaterialAnalysisLab` 与 `EnergyControlStation`
3. 在走廊上方或下方平行铺设 `TubularConnectorTunnel`，从 `EnergyControlStation` 引出，串至 `CoreVoidLab`、`MidSizeMaterialAnalysisLab`
4. 整个殖民地外围高地放置 1~3 座 `ObservationPostTelescope` 作为视觉边界与观测哨
5. 所有建筑基座 Y 坐标尽量对齐到同一水平线，桥段与管段以各自两端的铆钉/接头像素为"接缝锚点"重复贴图即可完成拼接

## 通用注意事项

- 所有素材皆带透明通道，边缘已抗锯齿处理，放置时无需额外裁剪
- 色调与锈蚀风格统一，混用不会出现明显违和
- `ConnectionBridgeSegment` 与 `TubularConnectorTunnel` 为段落贴图，其他四张为整体建筑贴图，代码中可按此区分加载策略
